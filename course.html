Materi Kuliah

Pengantar Web

Memperkenalkan dasar-dasar pengembangan web, termasuk HTTP, URL, dan arsitektur client-server. Contoh penggunaan: Menjelaskan cara kerja web browser dan server.

HTML

Mempelajari dasar-dasar HTML (HyperText Markup Language) untuk membuat struktur halaman web. Contoh penggunaan: Membuat halaman web sederhana dengan berbagai elemen HTML seperti paragraf, heading, dan tabel.

CSS

Mempelajari CSS (Cascading Style Sheets) untuk mendesain tampilan halaman web. Contoh penggunaan: Mengatur layout, warna, dan font halaman web agar lebih menarik dan responsif.

JavaScript

Mempelajari dasar-dasar JavaScript untuk menambahkan interaktivitas pada halaman web. Contoh penggunaan: Membuat fungsi validasi form dan menambahkan efek animasi pada elemen web.

PHP

Mempelajari dasar-dasar PHP (Hypertext Preprocessor) untuk pengembangan aplikasi web dinamis. Contoh penggunaan: Membuat sistem login sederhana dengan verifikasi pengguna.

Bootstrap

Mempelajari framework CSS Bootstrap untuk membuat halaman web yang responsif dan modern. Contoh penggunaan: Membuat tata letak grid dan komponen UI seperti navbar dan tombol dengan Bootstrap.

MySQL

Mempelajari MySQL untuk manajemen basis data relasional. Contoh penggunaan: Membuat dan mengelola database untuk aplikasi web, termasuk operasi CRUD (Create, Read, Update, Delete).

Materi yang Belum Dipahami

PHP

PHP masih menjadi tantangan karena kompleksitas dalam manajemen sesi pengguna, keamanan aplikasi, dan integrasi dengan basis data. Memahami cara efektif mengelola sesi pengguna untuk mengatur status login/logout dan data sesi menjadi kunci utama. Selain itu, diperlukan penerapan praktik keamanan seperti sanitasi input dan pengendalian akses untuk melindungi aplikasi dari serangan. Integrasi PHP dengan MySQL untuk melakukan operasi CRUD (Create, Read, Update, Delete) juga memerlukan pemahaman yang mendalam.

Aspek Deskripsi
Manajemen Sesi Pengguna Mengatur status login/logout dan data sesi pengguna.
Keamanan Aplikasi Praktik keamanan seperti sanitasi input dan pengendalian akses.
Integrasi dengan MySQL Operasi CRUD (Create, Read, Update, Delete) dengan MySQL.
